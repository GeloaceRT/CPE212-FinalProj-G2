About The Website

The practical exercises for the Automating Server Management course are highlighted in this E-Portfolio, which also shows the students' practical proficiency. The students utilized Ansible playbooks to automate server configurations, reducing excessive steps in software installation, network settings, and security protocols. They installed and configured essential tools such as OpenSSH for remote server connections, Elastic Stack (ElasticSearch, Logstash, Kibana) for logging and monitoring, Prometheus for system monitoring, and Nagios for alert management. The students worked with operating systems like Ubuntu Desktop and CentOS 7 to configure and manage servers, ensuring optimal system performance and security. The E-Portfolio includes detailed documentation, code snippets, and results, showcasing the student's ability to tackle complex server management tasks and their readiness for real-world IT-related challenges.
